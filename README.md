## Como executar o projeto

1. Clone o repositório do GitHub:
	```bash
	git clone git@github.com:designtechti490/landing-page
	```
2. Execute o projeto no Browser com o Live Server do VS Code:
	```bash
	ALT + L / Alt + O
	```
